# MIGHTY FOODS REST API Social Media

A full-stack, mobile friendly social media platform using Sprint Boot Maven. 

## Technologies:
- Backend: **Spring Boot**, **Maven**
- Frontend: **ReactJS**
- Middleware: **JWT**, **Websocket**, **JPA**
- Database: **MySQL**, **H2 Database Engine**
